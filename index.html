Color Mix

This div has a ratio color-mix background:

"background: color-mix(in srgb, red 50%, blue 50%);"

This div has a ratio transparent-mix background:

"background: color-mix(in srgb, rgba(255, 0, 0, 0.5) 50%, rgba(0, 0, 255, 0.5) 50%);"

This div has a ratio lightness-mix background:

"background: color-mix(in srgb, black plus 50% lightness, white minus 50% lightness);"

HSL Color Adjust (Chrome & Safari only)

This div has an HSL color-adjust background on OPACITY:

"background: hsl(from var(--pink) h s l / 0.3); padding: 1rem"

This div has an HSL color-adjust background on HUE:

"background: hsl(from var(--pink) calc(h + 10) s l / 1)"

This div has an HSL color-adjust background on SATURATION:

"background: hsl(from var(--pink) h calc(s * .5) l / 1);"
