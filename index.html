The purpose of subgrid is to allow a nested grid (a grid inside another grid item) to align its grid lines with the parent grid. This is useful when you want the content within a grid item to line up with the rest of the items in the grid.

In this example, the .parent-grid sets up a two-column layout. The second grid item contains another grid, .child-grid. Instead of defining its own grid layout, .child-grid uses subgrid to line up with the .parent-grid's columns. This means that the .subgrid-item elements will align under the second column of the .parent-grid.

The subgrid is especially useful when you want nested items to align with the main content or when you have complex nested structures and want to maintain a clean, consistent layout without manually syncing the grid gaps, column sizes, and row sizes of the nested grid with its parent.
